Instalación:   
El repositorio se puede descargar directamente desde Github o clonarlo en la terminal de comandos
mediante el comando: git clone https://github.com/pato454/Analizador-Sintaxis.git

Instrucciones de Uso:  
En el repositorio se enuentran los 4 ejemplos de inputs, el archivo Reglas.txt que contiene la gramatica asignada
y la carpeta "Analizador" que contiene todo el código del proyecto. Para probar cualquiera de los cuatro ejemplos, se debe 
seguir la siguiente serie de pasos:
  - Copiar el texto de cualquiera de los cuatro ejemplos.
    <img width="506" height="278" alt="image" src="https://github.com/user-attachments/assets/2cb14be0-804d-4bc7-90b4-7c00fb6069ad" />
  
  - Ir a la carpeta "Analizador" y pegar el texto en el archivo input.txt (por defecto tiene el texto del archivo
    input_correcto_1.txt).

    <img width="499" height="290" alt="image" src="https://github.com/user-attachments/assets/e56fe25e-13c3-48df-ac5d-3d109686f6ab" />

    
  - Abrir la terminal de VSCODE, moverse a la carpeta "Analizador" y ejecutar el comando "node index.js".
  <img width="341" height="22" alt="image" src="https://github.com/user-attachments/assets/7d38f51f-f500-481f-ba7e-d438ba3da17a" />


Para probar otro ejemplo simplemente se repiten dichos pasos.
